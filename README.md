# Correlates of War (COW) Data Analysis

## Overview

This project explores historical conflict and international relations data using the Correlates of War (COW) dataset. The analysis examines patterns in conflict occurrences, trends over time, and relationships between key variables related to war and state interactions.

## Dataset

The dataset used in this project is the Correlates of War dataset, obtained from the correlates of war website:

https://correlatesofwar.org/data-sets/

Unlike some of my other projects, the dataset is included in this repository.

## Project Goals

* Analyze patterns in international conflicts
* Explore trends across time periods
* Identify relationships between variables in the dataset
* Visualize key findings through charts and graphs
